Step 3:

Specify Inputs

Below are some toggleable elements, based on the original deep music visualizer.
If you don't know what these do, then just ignore them and click next at the bottom!

Duration:The duration of the song. If not selected, will process the entire song.
Pitch Sensitivity: How rapidly the class vector responds to changes in pitch. Higher represents more sensitive. 220
Tempo Sensitivity: How rapidly the noise bector changes with volume and tempo. Higher represents more sensitive. 0.25
Depth:Max value of vector units. Closer to 1 is more thematically rich, closer to 0 is "deeper" structures (like dogs and faces). 1
Number of Classes: The number of image classes that will be handled simultaneously. 12
Jitter: Adds random noise. 0.5
Frame Length: Duration of frames -- inversely proportional to frame rate. Lower values represent higher quality.
Truncation: Controls the variability of the images generated by limiting the max values. 1 allows all, 0 are simpler images with more normal looking objects. 1
Smooth Factor: There is a smoothing process using interpolation. Useful for fast music, which one should lower the smooth factor. 20
Subtitles: Include subtitles?
