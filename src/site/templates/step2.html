Welcome to the GAN Music Visualizer!

This visualizer will create videos similar to those seen in the following places:
YouTube
TikTok
Instagram

To get started, all you will need is an MP3 file for the song you are trying to visualize, as well as the lyrics. The following prompts will walk you through how to use them.



Step 2:

Upload a .LRC File With The Lyrics

To do this, use the LRC Generator. This will generate a LRC file, which encodes the timing information
for the lyrics. If you would like to not use lyrics, you can simply type a word that you wish the visualizer to use to visualize.
If you want, you can use any text, used at any point in the song (it doesn't have to be lyrics), and this will inform the images.
